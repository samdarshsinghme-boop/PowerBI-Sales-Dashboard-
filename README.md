# 📊 Sales Management Dashboard (Power BI)

This project is an interactive Sales Management Dashboard built using Power BI to analyze and visualize business performance.

## 🔹 Overview
The dashboard provides a comprehensive view of sales, profit, and customer behavior across different regions, categories, and time periods.

## 🔹 Key Features
- 📈 Total Sales, Profit, Quantity, and Orders Overview  
- 🌍 Region-wise Sales Analysis (Central, East, South, West)  
- 📊 Monthly Sales & Profit Trends (Year-over-Year Analysis)  
- 🛒 Sales by Category and Sub-Category  
- 🚚 Sales Distribution by Ship Mode  
- 💳 Payment Mode Analysis  
- 📍 State-wise Sales & Profit Mapping  

## 🔹 Insights Generated
- Identified top-performing regions and categories  
- Analyzed monthly growth trends and seasonal patterns  
- Compared profit distribution across different segments  
- Evaluated customer purchasing behavior and shipping preferences  

## 🔹 Tools & Technologies Used
- Power BI  
- Data Visualization  
- Data Analysis
